## 📋 Описание проекта

**Training Planner** - это десктопное GUI-приложение для планирования, отслеживания и анализа тренировок. Приложение позволяет пользователям вести учет своих спортивных занятий, фильтровать их по различным критериям и сохранять данные в JSON формате.

### Основные возможности:

- 📝 **Добавление тренировок** с указанием даты, типа и длительности
- 📊 **Табличное отображение** всех тренировок с удобной навигацией
- 🔍 **Фильтрация записей** по типу тренировки и диапазону дат
- 💾 **Сохранение и загрузка данных** в формате JSON
- 📈 **Отображение статистики** (общее количество, суммарная длительность, средние показатели)
- ✏️ **Редактирование и удаление** существующих записей
- 📤 **Экспорт данных** в отдельные JSON файлы
- ✅ **Валидация вводимых данных** (формат даты, положительная длительность)
